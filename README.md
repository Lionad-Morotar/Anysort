# any-sort

 强大、优雅、无任何依赖的排序方法。
 
 你可以以一个或多个属性、长度比较、数值比较、字符串比较等方法进行排序。更加惊奇的是，这些方法可以任意组合。示例文件请看 [examples 文件夹](./examples/)。
